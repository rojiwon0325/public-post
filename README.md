## 요구사항 분석

1. 게시글 생성

- 제목(최대 20자), 본문(최대 200자)으로 구성
- 비밀번호 설정(숫자를 포함한 6자 이상의 문장)

2. 게시글 수정/삭제

- 비밀번호를 통한 인증

3. 게시글 조회

- 최신순 조회
- pagination(20개씩) 적용
